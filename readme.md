# Virtue TM — To-Do App with Revision Timer

A simple and minimal **To-Do List Web App** built using **HTML, Tailwind CSS, and JavaScript** as a revision project before moving into **Node.js** and **React**.

The project includes:

- ✅ Dynamic task creation
- ✅ Task removal using event delegation
- ✅ 25-minute productivity timer
- ✅ Start / Pause / Reset functionality
- ✅ Responsive Tailwind CSS styling
- ✅ Font Awesome icons integration

---

# 🚀 Features

## 📝 To-Do List

- Add tasks dynamically
- Tasks are rendered instantly on the screen
- Click on a task to remove it
- Uses **DOM manipulation** and **event delegation**

## ⏳ Productivity Timer

- 25-minute countdown timer
- Start, Pause, and Reset controls
- Alerts the user when the timer ends
- Built using:
  - `setInterval()`
  - `clearInterval()`
  - Event listeners

## 🎨 UI

- Built completely using **Tailwind CSS**
- Minimal and clean design
- Fixed productivity timer section
- Responsive behavior for smaller screens
